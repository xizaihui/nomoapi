### 🐛 生产环境缓存问题排查 (晚间)
- 用户报告生产环境侧栏仍显示 "日志" 而非 "审计"
- Ctrl+Shift+R 和无痕模式均无效
- **根因**: Go embed 在 Docker build 阶段将 dist/ 嵌入二进制。之前部署流程 SCP dist 到服务器后 docker build，但 `COPY . .` 阶段 Go build 使用的 dist 是工作树内的文件 — 理论上应该是新的
- 最终在生产服务器重新执行 `docker build --no-cache -t newapi-aurora:latest .` + `docker compose up -d --force-recreate --no-deps new-api`
- 重建后 prod ✅ 200，等待用户确认
- **教训**: `--no-cache` 对 Go embed 场景很重要，Docker layer cache 可能导致旧 dist 被复用
