## Fixed Stale Dist / Dynamic Import Error (~22:02 UTC)
- **Symptom**: Clicking sidebar navigation caused "Failed to fetch dynamically imported module: index-CVqbowfk.js"
- **Root cause**: Old dist files mixed with new build — `index.html` referenced new chunks but old lazy-loaded chunks from previous build still present, causing hash mismatches
- **Fix**: `rm -rf dist node_modules/.vite` → clean `bun run build` → `docker build --no-cache` → `docker compose up -d --force-recreate --no-deps new-api`
- **Lesson**: ALWAYS `rm -rf dist` before `bun run build` to prevent stale file mixing. This is a recurring issue.
