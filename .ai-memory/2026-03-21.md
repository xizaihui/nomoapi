### New Typography Scale
| Level | Usage | Size | Weight |
|-------|-------|------|--------|
| H1 | Page titles | text-xl (20px) | font-semibold |
| H2 | Section/modal titles | text-base (16px) | font-medium |
| Body | Content | text-sm (14px) | font-normal |
| Caption | Hints, table headers | text-xs (12px) | font-medium |
| Stat | Big numbers | text-2xl (24px) | font-semibold |

### New Color Hierarchy (3 levels, down from 10)
- L1: `text-foreground` — titles, primary content
- L2: `text-foreground/80` — secondary content, table data
- L3: `text-muted-foreground` — captions, hints, disabled

### Key changes
- Eliminated 101 `text-slate-*` hardcoded references → semantic tokens
- Eliminated 8 scattered opacity levels (/35/40/45/55/60/65/70/75) → consolidated to /80 only
- SideSheet title: inline `fontSize:15px fontWeight:500` → `text-base font-medium` Tailwind class
- Table header: `text-[11px]` custom size → `text-xs` standard
- Typography.Title h1-h6: remapped to new scale (h1=text-2xl, h2=text-xl, h3=text-base, etc.)
- `font-bold` → `font-semibold` for stats/headings (only 1 bold left: red price emphasis)
- Global `--radius`: 0.5rem → 0.625rem (rounder feel)
- Sidebar nav border-radius: 6px → 8px
- Added `letter-spacing: -0.011em` to body (Inter font optimal)
- Sidebar selected weight: 600 → 500 (medium, more refined)
